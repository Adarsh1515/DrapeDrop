How to Test the Flow:

  1. Start Registration: Click "Start Registration" to go to the auth page
  2. Fill Form: Complete the registration form with your details
  3. Phone Verification: You'll be redirected to phone verification page
  4. Enter OTP: Enter the 6-digit code sent to your phone
  5. Complete Registration: After verification, you'll be redirected to login
  6. Login: Use your credentials to log in to the system

Important Notes:

  • Use a real phone number for OTP testing
  • Ensure Firebase is properly configured
  • Phone number should include country code (e.g., +91)
  • OTP will be sent via SMS (real Firebase service)
  • Check your phone's SMS inbox for the verification code
